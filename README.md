### 🧠 Project Overview
**Acoustic Stream Orchestrator (ASO)** is an advanced, high-performance virtual audio driver and soundboard suite designed for modern broadcasters and competitive communication. By implementing a custom kernel-level audio bridge, ASO allows for seamless injection of high-fidelity audio samples directly into your microphone stream with near-zero latency. Built for the **2026 broadcast standards**, it ensures crystal-clear playback and robust signal routing.

### 🚀 Key Capabilities
* **🎙️ Virtual Signal Injection:** Directly route high-quality WAV/FLAC samples into any voice application (Discord, Teams, Games).
* **⚡ Ultra-Low Latency:** Optimized buffer management ensuring <5ms delay between trigger and output.
* **🎹 Global Hotkey Engine:** Assign complex sound sequences to any key combination with system-wide focus.
* **📈 Dynamic Gain Normalization:** Automated volume leveling to ensure samples match your microphone’s decibel range perfectly.
* **🛠️ High-Fidelity Encoder:** Native support for 24-bit/192kHz audio processing for professional-grade output.

### 📦 Installation
1. Ensure your system supports **Kernel Audio Streaming**.
2. Download the `Acoustic_Orchestrator_Setup.msi` from the section below.
3. Install the virtual audio driver when prompted by the installer.
4. Launch the Dashboard and select your primary input (Microphone) and output devices.
5. Drag and drop your audio library into the workspace to begin orchestrating.

* **Driver Layer:** Custom Virtual Audio Cable (VAC) Architecture
* **UI Framework:** Qt 6 / QML (Modern Dark Aesthetics)
